# Contexto do Sistema
## Visão Geral
Este projeto é um backend de gestão acadêmica construído utilizando XanoScript e Spec-Driven Development (Desenvolvimento Orientado por Especificações).

## Propósito
Permitir que os usuários gerenciem disciplinas acadêmicas e dados relacionados.

## Entidades Principais
- Usuários (Autenticação nativa do Xano)
- Disciplinas (Matérias acadêmicas vinculadas aos usuários)

## Relacionamentos
- Cada disciplina pertence a um único usuário.

## Evolução Esperada
O sistema incluirá, futuramente, APIs, tarefas de automação e integrações.
